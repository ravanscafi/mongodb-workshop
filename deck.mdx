# Aplicação

```
git clone \
    git@github.com:ravanscafi/mongodb-workshop-base.git
cd mongodb-workshop-base
```

---

```bash
cp .env.example .env
docker-compose up -d
docker-compose exec web php artisan key:generate
docker-compose exec web composer install
```

e abrir o [localhost](http://localhost)
